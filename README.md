# Simulador de dados :game_die:

Um simples simulador de dados via shell do Windows usando a linguagem python.

## Como utilizar?

-  Abra o seu terminal do windows e utilize o seguinte comando: python simula_dado.py ou python3 simula_dado.py dependendo de como o seu SO estiver reconhecendo o python via Command Line.
- Irá abrir uma tela descritiva, então só seguir o passo a passo que ele solicitar.



### Avisos

- O Python precisa estar instalado  para rodar via terminal.
